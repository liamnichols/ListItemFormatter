Mode

@objc(LNListItemFormatterMode)
public enum Mode : Int

The following constants specify predefined format modes for list items.

The mode is used in conjuction with the Style to pick the most appropriate set of patterns used by the formatter. You can read more about the patterns in the Unicode documentation.
